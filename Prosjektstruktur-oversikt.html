# 🗂️ Prosjektstruktur – Butikkoversikt.no

En organisert oversikt over prosjektets mapper, filer og hvordan de henger sammen.

## 🌐 HTML-sider
- index.html → Forside
- kategori.html → Viser valgt kategori + underkategori-filter
- kategori-mal-utforsk.html → Utforsk alle kategorier med meny
- alle-butikker.html → Full liste
- ukens-anbefalte.html → Ukens utvalgte
- kampanjer.html → Kampanjeside
- guider/main-guider.html → Guideoversikt
- guider/guide-*.html → Enkeltguider

## 🎨 CSS-stilark (i style.css via @import)
- root.css, header.css, footer.css
- guide-card.css, top25-card.css, promo-card.css
- responsive.css, theme.css

## 📊 JSON-data
- butikker.json, kampanjer.json, guider.json, annonser.json

## ⚙️ JS-filer
- generate-kategori.js → kategori.html
- generate-kategorioversikt.js → kategori-mal-utforsk.html
- generate-guider.js → main-guider.html
- generate-kampanjer-forside.js → kampanjer på forside
- autocomplete-søk.js, theme-toggle.js, main.js → støttefiler
