Admin-skript: Brukerveiledning og rekkefølge

✅ Før du starter

  1. Installer nødvendige avhengigheter (kun én gang):
    .\.venv\Scripts\python.exe -m pip install requests pillow
  2. Start lokal server for admin-panelet:
    .\.venv\Scripts\python.exe server.py
    Gå deretter til http://localhost:8000 i nettleseren.
