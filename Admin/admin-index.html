🛠️ Adminpanel – Verktøyoversikt

📦 Butikkredigering

  • ✏️ Rediger butikker én og én Endre navn, URL, affiliate, tags, kategori og mer for én butikk av gangen.
  • 🏷️ Tag-verktøy (automatisk tagging) Foreslår tags basert på underkategori. Lar deg rydde eller redigere tags manuelt.
  • 🔗 Importer affiliate-URLer (CSV) Lim inn CSV-liste med navn og affiliate-lenke for å oppdatere flere butikker raskt.
  • 💰 Affiliate-status oversikt Gir deg oversikt over hvilke butikker som er affiliate, partner eller mangler lenke.
  • ✅ Validator for duplikater og mangler Oppdager duplikater, tomme felter og feil i `butikker.json`.

🖼️ Bilder og struktur

  • 🖼 Finn manglende bilder Sjekker hvilke butikker som mangler `.webp`-logo eller har feil bildebane.
  • 📂 Rediger kategori/underkategori Flytt butikker mellom kategorier eller legg inn manglende strukturinformasjon.
  • 🔧 Lag slugs til bildefilnavn Konverter butikknavn til filvennlige navn som kan brukes til bilde-URL.

📢 Metadata og kampanjer

  • 🔍 SEO: tittel + meta-beskrivelse Legg til `seoTitle` og `seoDesc` som brukes i ` `-metadata på nettsiden.
  • 🌟 Ukens anbefalte butikker Merk butikker som anbefalt og skriv en kort begrunnelse som vises på forsiden.
  • 📢 Kampanjer (JSON validering) Lim inn eller test kampanjestruktur i JSON. Brukes for kampanjeseksjoner.
  • 📢 Kampanjer (JSON validering) Legg til nye kampanjer i JSON format. Brukes for kampanjeseksjoner.

🧠 Avanserte funksjoner

  • 🔎 Avansert søk og filtrering Globalt søk i alle butikker (grunnstruktur for videre utvikling).
  • 🤖 AI-forslag til tags Viser eksempel på hvordan man kan generere relevante tags fra tekstbeskrivelse.
  • ✏️ Batch-redigering via JSON Lim inn flere butikker i JSON og gjør masseendringer samtidig.
