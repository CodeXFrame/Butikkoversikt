🧭 Lanseringsplan for Butikkoversikt.no

🚀 1. Klargjøring av innhold

🏣 Nettbutikker

  • Gå gjennom alle kategorier og fyll inn minst 10–15 butikker per kategori
  • Kontroller riktige subcategory, tags, alt, description, image og url
  • Fjern duplikater og test alle lenker

📘 Guider og tips

  • Legg til relevante guider med bilde, tittel, beskrivelse og lenke
  • Kontroller at guider lenker til riktig kategori eller ekstern side

💸 2. Kampanjer og Ukens anbefalte

🎯 Kampanjer

  • Erstatt dummy-data med faktiske kampanjer i kampanje-JSON
  • Legg til bannerbilder og lenker
  • Koble relevante guider til kampanjesider

⭐ Ukens anbefalte

  • Velg 4–6 butikker manuelt eller basert på kampanjer

💼 3. Affiliate- og inntektsmodell

  • Identifiser hvilke butikker som tilbyr affiliate-programmer
  • Registrer deg i relevante nettverk: Adtraction, Adservice, Awin osv.
  • Bytt ut url med affiliate-lenker der det er mulig
  • Merk affiliate-lenker ved behov (f.eks. ikon eller tekst)

🔮 4. Kvalitetssikring og ytelse

  • Test alle knapper, filtre og mobilvisning
  • Komprimer bilder til .webp eller .avif
  • Legg til alt-tekst, meta tags og beskrivelse
  • Lag sitemap.xml og robots.txt

🗓 5. Lansering og vedlikehold

  • Publiser og test hele nettsiden via Netlify og domenet
  • Legg inn "Hold meg oppdatert"-funksjon
  • Promoter via forum, grupper og nyhetsbrev
  • Sett opp Trello/Notion-tavle for ukentlig vedlikehold

📦 Bonusideer

  • Blogg med trendartikler
  • "Finn gave"-velger basert på budsjett/anledning
  • Statistikk på mest klikkede guider og butikker
  • Nyhetsbrev med ukens kampanjer
